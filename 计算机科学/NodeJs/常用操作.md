## 项目管理

### 创建项目

```bash
mkdir my-project && cd my-project
pnpm init
```

### 示例：创建原生TypeScript项目**

```bash
pnpm add -D typescript
pnpm add -D @types/node
pnpm add -D vite

npx tsc --init
```

**全局包**

查看已安装的全局包

```bash
npm list -g --depth=0
```

查看过时的全局包

```bash
npm outdated -g
```

更新所有全局包

```bash
npm update -g
```
