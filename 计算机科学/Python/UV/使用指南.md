## UV 项目结构

```
example
├── .python-version
├── README.md
├── main.py
└── pyproject.toml
```

## 基本操作

**创建项目**

创建 example 项目目录

```bash
uv init example
```

**添加/移除包**

添加 numpy 包

```bash
uv add numpy
```

移除 numpy 包

```bash
uv remove numpy
```

**更新项目环境**

```bash
uv sync
```

**更新 lockfile**

```bash
uv lock
```

**运行**

```bash
uv run main.py
```
