# Gitea 使用技巧

## 1. 通用 CI 流

1. 善用 Github/Gitea 的 `Environments` 与 `Secrets`
2. 善用 workflows 的短路运算，如 `${{ vars.DOCKER_REGISTRY || 'ghcr.io' }}`，当定义了 `vars.DOCKER_REGISTRY` 时则使用该值，否则使用 `ghcr.io`

## 2. 多远程仓库推送

```bash
git remote set-url --add --push origin git@github.com:hiromuraki/sample_git.git
git remote set-url --add --push origin http://localhost:3000/test/sample.git
git branch -M main
git push -u origin main
```
